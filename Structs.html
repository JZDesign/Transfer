  • TransferElement

    The implementation of the Transferable Protocol

    This structure is made to contain all of the details about each HTML element and will render as HTML when needed.

    Usage:

     TransferElement(
     element: .img,
     attributes: [
         .src(value: "images/cute_kitten.jpeg"),
         .alt(value: "A cute kitten")
         ],
     content: ""
     ).rendered()

    This returns:

     "<img src=\"images/cute_kitten.jpeg\" alt=\"A cute kitten\"></img>"
