  • Attribute

    A list of common HTML Element Attributes i.e. id, style, target, class, alt, etc.,

    Usage:

    Passing attributes to the TransferElement will automatically get rendered appropriately in the HTML.

    TransferElement(
    element: .img,
    attributes: [
        .src(value: "images/cute_kitten.jpeg"),
        .alt(value: "A cute kitten")
        ],
    content: ""
    )

    This example renders the src and alt attributes inside of an Image element. That result would look like this:

    <img src="images/cute_kitten.jpeg" alt="A cute kitten"></img>
